Acerca del Bootcamp

Este Bootcamp 100% enfocado en Colombia está diseñado para desarrolladores Web2 que quieren dar el salto al ecosistema Web3.

Aprenderás construyendo: nada de teoría innecesaria, todo el contenido está en GitHub para que tengas acceso abierto, colaborativo y actualizado.

Cada clase se basa en proyectos funcionales usando herramientas como cCOP, Karma GAP y Proof of Ship.

Impulsado por Celo Colombia y sandusky.eth, este bootcamp es tu puerta de entrada al mundo DeFi con impacto y propósito local.

Agenda del Bootcamp

Sesiones en vivo, 2 horas cada una, 80% hands-on, 20% walkthrough.

Sesión 1: Hello Celo + Setup de Proof of Ship

Despliegue inicial de contrato y creación del proyecto en Karma GAP.

Sesión 2: Vault con Time Lock

Contrato inteligente y UI para bloquear CELO o cUSD por tiempo definido.

Sesión 3: Interfaz de Multi-Sig para cCOP

Frontend que permite firmar colectivamente transacciones de cCOP.

Sesión 4: Dashboard de Uniswap en Celo

Visualización de data de pools usando SDK de Uniswap y viem.

Al registrarte, aceptas cumplir con estas normas para garantizar una experiencia positiva para todos.
